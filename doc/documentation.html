安装

~ $ gem install jekyll
~ $ cd devoll
~/devoll $ jekyll serve
# => Now browse to http://localhost:4000

结构

这个体系建立在科学研究的基础上 foundation framework, to understand what the "row" and "column" classes mean, please refer to the foundation documentation.
